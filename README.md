# covid-streamlit

This repository contains `covid.py`, a Streamlit dashboard that visualizes WHO COVID-19 global data.

Files:

How to run locally:

1. Create a virtual environment and activate it.
2. Install dependencies: `pip install -r requirements.txt`.
3. Place `WHO-COVID-19-global-data.csv` in the same directory (or update the path in the code).
4. Run: `streamlit run covid.py`
